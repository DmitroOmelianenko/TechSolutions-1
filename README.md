# TechSolutions-1
ЗАПУСКАЄМО КОД ЛОКАЛЬНО

1. Заванатажте програму VScode для Windows/Linux/Mac
2. Завантажте програму GitHub Desktop 
3. Авторизуйтесь в програмі GitHub Desktop через GitHub у браузері
4. Склонуйте репозиторій натиснувши code -> open with github desktop в браузері у вкладці code
5. Завантажте плагін live Server в програмі VScode
6. Перейдіть у файл index.html
7. В низу зправа є кнопка Live Server, натисніть її. І вас автоматично перекине в браузер
